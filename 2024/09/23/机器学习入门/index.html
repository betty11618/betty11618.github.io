机器学习入门

Posted on 2024-09-23

入门

一文看懂机器学习「3种学习方法+7个实操步骤+15种常见算法」 (easyai.tech)

机器学习不是某种算法，而是很多算法的统称。

深度学习就是其中的一种算法

基本思路

将问题转化为数学问题，然后解答数学题

最为困难的部分是抽象

实现原理

训练集

特征

建模

模型

机器学习：通过训练集，不断识别特征，不断建模，最后形成有效的模型

分类

根据训练方法分：

  • 监督学习
  • 非监督学习
  • 强化学习

监督学习

给算法一个数据集，并给出正确答案

机器直接学习正确答案的计算方法

优劣势

  • 学习效果好
  • 成本高

特点

需要有明确的目标，清楚想要的结果

流程

  1. 选择适合目标任务的数学模型
  2. 将一部分已知的“问题和答案”（训练集）给机器去学习
  3. 机器总结出自己的方法论
  4. 将“新的问题”（测试集）给机器解答

任务

  • 回归：预测连续的、具体的数值
  • 分类：对各种事物分门别类，用于离散型预测

举栗子

回归

  1. 构建问题，选择模型
    • 找影响因素来构建模型
  2. 收集已知数据
    • 分数据
  3. 训练出理想模型，验证公式
    • 将这五种数据套入公式，计算出信用分
    • 用计算出来的信用分跟实际的信用分（预先准备好的）进行比较
    • 评估公式的准确度，如果问题很大再进行调整优化
  4. 对新用户进行预测

分类

  1. 构建问题，选择模型
  2. 收集已知数据
  3. 训练出理想模型
  4. 对新用户进行预测

主流的监督学习算法

非监督学习

给定的数据集没有 正确答案 ，所有的数据都是一样的。

任务

从给定的数据集中，挖掘出潜在的结构

将不打标签的照片给机器

机器通过学习会将照片分为两类，但是机器不知道哪个是猫哪个是狗 ，只是分为两类

对比监督学习

监督学习 无监督学习
目的明确的训练方法 没有明确目的的训练方法，无法提前知道结果是什么
给数据打标签 不需要给数据打标签
可以衡量结果 几乎无法量化效果如何

无监督学习是一种机器学习的训练方式，它本质上是一个统计手段，在没有标签的数据里可以发现潜在的一些结构的一种训练方式。

特点

  • 没有明确的目的
  • 不需要给数据打标签
  • 无法量化结果

使用场景

  • 案例一：发现异常，分类速排出正常用户，更有针对性的对异常的洗钱行为进行深入分析
  • 案例二：用户细分，通过用户行为对用户进行分类
  • 案例三：推荐系统，淘宝根据浏览行为进行分类，发现购买行为相似的用户，推荐这类用户喜欢的商品

常见的两类无监督学习算法

  • 聚类：一种自动分类的方法，不清楚聚类后的几个分类每个代表什么意思
  • 降维：看上去很像压缩，这是为了在尽可能保存相关的结构的同时降低数据的复杂度

聚类算法：k均值聚类

聚类算法：层次聚类

适用于不知道应该分几类的情况

层次聚类会构建一个多层嵌套的分类，类似一个树状结构

步骤

降维算法：主成分分析 - PCA

主成分分析：把多指标转化为少数几个综合指标

主成分分析经常用 减少数据集的维数，同时保持数据集的对方差 贡献最大的特征 。这是通过保留低阶主成分，忽略高阶主成分做到的。这样低阶成分往往能保留住数据的最重要方面。

变换的步骤

降维算法：奇异值分解 - SVD

线性代数中的一种重要的矩阵分解，奇异值分解则是特征分解 在任意矩阵上的推广

主流算法

两大分类，主要差异为 智能体是否能完整了解或学习到所在环境的模型

分类 免模型学习 有模型学习
优点 对环境有提前的认知，可以提前考虑规划 更加容易实现，容易在真实场景下调整到很好的状态
缺点 如果模型跟真实世界不一致，在实际使用场景下表现得不好 效率不如免模型学习

免模型学习方法更受欢迎，得到更加广泛的开发和测试

免模型学习：策略优化

免模型学习：Q-Learning

有模型学习：纯规划

有模型学习：Expert Iteration

其他分类

  • 基于概率 - 基于价值
  • 回合更新 - 单步更新
  • 在线学习 - 离线学习

实操步骤

  1. 收集数据 —> 数据的数量和质量直接决定预测模型的好坏
  2. 数据准备 —> 训练集、验证集、测试集
  3. 选择一个模型
  4. 训练
  5. 评估 —> 指标：准确率、召回率、F值
  6. 参数调整
  7. 预测（开始使用）
