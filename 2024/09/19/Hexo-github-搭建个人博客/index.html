Hexo + github 搭建个人博客

Posted on 2024-09-19

安装node.js

Node.js — 在任何地方运行 JavaScript (nodejs.org)

注意环境变量配置（安装时自动给配置了，不需要手动配）

检查安装是否成功

github创建仓库

仓库命名 xxx.github.io

进入仓库，创建文件 index.html

git for windows安装

Git for Windows

github ssh配置

使用 git 里面的 git-bash 输入

找到.ssh文件夹，打开id_rsa.pub文件

将刚刚配置好的ssh，添加到自己的github账号中

Hexo本地使用

Hexo

创建一个新的文件夹，进入

输入命令行下载hexo

验证证书失败，但通过复制数据的方式完成了初始化

启动服务器，端口为5555

本地下载完成

sublime安装

（其实芝士需要一个文本编辑器notepad++也行

Sublime Text - Text Editing, Done Right

打开 Sublime Text 将 Hexo 下载所在的文件夹拖进去

保存

将Hexo发布到github上

打开本地Hexo的配置文件

之前已经使用SSH进行github部署了，修改配置文件中 repo 的地址为使用SSH

此处填写自己的网址

生成，提交

发现报错，查看Hexo文档后发现是缺少依赖

清除缓存，生成，提交

即可

需要注意的是，github仓库是否启用了 github pages

最后我们尝试打开网站

成功发布
